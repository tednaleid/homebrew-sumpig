# homebrew-sumpig

Homebrew tap for [sumpig](https://github.com/tednaleid/sumpig), a Merkle tree directory fingerprinting and comparison tool.

## Install

```bash
brew install tednaleid/sumpig/sumpig
```

Or:

```bash
brew tap tednaleid/sumpig
brew install sumpig
```

## Update

```bash
brew upgrade sumpig
```
